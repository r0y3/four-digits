Problem
=======

Given four digits find the maximum valid time that can be displayed on a digital clock(24 hours format) using those digits.

Run tests
=========

```bash
git clone https://github.com/r0y3/max-valid-time.git
cd max-valid-time
python -m unittest discover
```
